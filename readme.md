## ğŸ”§ Installation

**For Datapack + Resource Pack:**
1. Download both the **Datapack** and **Resource Pack** files
2. Place the datapack ZIP in your world's `datapacks/` folder
3. Place the resource pack ZIP in your `resourcepacks/` folder and enable it in-game
4. Reload with `/reload` or restart your world

**For Mod Loaders (Fabric/Forge/NeoForge/Quilt):**
- Download the `.jar` file and place it in your `mods/` folder
- *Requires Fabric API for Fabric/Quilt*
